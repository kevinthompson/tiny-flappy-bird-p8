# Tiny Flappy Bird

A tiny game made at 64x64 resolution using [object-oriented programming](https://www.youtube.com/watch?v=X9qKODb-wXg) in [PICO-8](https://www.lexaloffle.com/pico-8.php). It's available to play on Itch.io: https://kevinthompson.itch.io/tiny-flappy-bird.
